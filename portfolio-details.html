Intelligent Learning Management System

An 8 week project

  1. Home
  2. Portfoio Details

A content management system (CMS) is an application that is used to manage web content, allowing course instructors to create, edit and delete. Content in a CMS is typically stored in a database and displayed in a presentation layer based on a set of templates.
Our system is divided into two parts, A content management application (CMA) – this is the part that allows instructors to add and manage content.
A content delivery application (CDA) – this is the backend, behind-the-scenes process that takes the content provided in the CMA, stores it properly, and makes it visible to backend support.
The features in the system are to create content, generate JSON from the given inputs, storing the JSON data in the server, previewing the content, and deleting the content.
